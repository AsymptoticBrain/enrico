Enrico
======

Enrico helps you with your Fermi data analysis.

Produce spectra, maps and lightcurves for your target 
simply by editing a config file and running a python script.

Browse the documentation and run the tutorial analysis
to get started.

Please give us feedback by filing bug reports and feature
requests on github or by asking questions on the mailing list.
